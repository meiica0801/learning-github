## learning-github

GitHub の使い方を、自分用にゆっくり・しっかり学ぶためのログ用リポジトリです。

### このリポジトリでやること

- **Git / GitHub の基本操作の練習**
  - `git init`, `git add`, `git commit`, `git push` などのコマンド練習
  - ブランチを切る / マージする / コンフリクト解消の体験
- **GitHub 上での操作に慣れる**
  - リポジトリの作成・クローン
  - Pull Request (PR) の作成とレビューの流れを練習
  - Issues を使ったタスク管理の練習
- **学んだことのメモ**
  - よく使うコマンド
  - 「ハマったところ」と「解決方法」
  - 自分なりのベストプラクティス

### 想定している使い方

- 新しく覚えたコマンドや手順を試すとき、このリポジトリで自由に実験する
- うまくいった／いかなかったことを Markdown ファイルなどにメモして残す
- 時々 README やメモを読み返して、自分の成長を振り返る

### ログの残し方のアイデア

- **日付ごとのメモを作る**
  - 例: `logs/2026-02-10-git-basic.md` のようなファイル名で学びを保存
- **テーマ別にまとめる**
  - 例: `docs/branching.md`, `docs/conflict-resolution.md` など
- **コマンドチートシートを作る**
  - よく使うコマンドだけを一覧にした `git-cheatsheet.md` を置いておく

### これからの拡張アイデア

- GitHub Actions の簡単なワークフローを作ってみる
- コマンド実行例を含んだチュートリアル的なドキュメントを追加する
- 自分用の「Git / GitHub 学習ロードマップ」を書いてみる

---

難しいことを一気にやろうとせず、少しずつ「今日はこれだけできた！」を増やしていくための場所として使っていきます。
